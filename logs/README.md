# Mirror-gen System Logs Directory
# 미러-젠 시스템 로그 디렉토리

This directory contains log files generated by the Personal AI Optimization System.

## Log Files / 로그 파일

- `backend.log` - Backend server logs / 백엔드 서버 로그
- `frontend.log` - Frontend development server logs / 프론트엔드 개발 서버 로그  
- `mirror_gen_system.log` - System-wide logs / 시스템 전체 로그
- `optimization_*.log` - AI optimization process logs / AI 최적화 프로세스 로그
- `warmup_*.log` - Navier-Stokes warmup logs / 나비에-스토크스 웜업 로그
- `quantum_*.log` - Quantum preparation logs / 양자 준비 로그
- `reflex_*.log` - Reflex controller logs / 반사 제어기 로그

## Log Levels / 로그 레벨

- `DEBUG` - Detailed diagnostic information / 상세 진단 정보
- `INFO` - General information / 일반 정보  
- `WARNING` - Warning messages / 경고 메시지
- `ERROR` - Error messages / 오류 메시지
- `CRITICAL` - Critical system errors / 치명적 시스템 오류

## Log Rotation / 로그 회전

Logs are automatically rotated to prevent excessive disk usage. Old logs are compressed and archived.

로그는 과도한 디스크 사용을 방지하기 위해 자동으로 회전됩니다. 오래된 로그는 압축되고 보관됩니다.

## Viewing Logs / 로그 보기

```bash
# View recent logs
tail -f logs/mirror_gen_system.log

# View all logs
cat logs/backend.log

# Search for errors
grep -i error logs/*.log

# Follow multiple logs
tail -f logs/*.log
```
